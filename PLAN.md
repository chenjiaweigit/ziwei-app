### 用户系统

**位置：** `lib/db.ts` + `app/api/auth/*` + `lib/auth.ts`

- 手机号注册/登录（无密码，无验证码）
- JWT 认证（30 天有效期）
- JSON 文件数据库 (`data/db.json`)
- **无前端页面** — 没有登录/注册 UI，没有 `AuthProvider`

### 会员体系

| 功能 | 免费 | Pro (¥29.9/月) | 终身 (¥99.9) |
|------|------|---------------|-------------|
| 排盘 | 不限 | 不限 | 不限 |
| AI 解盘 | 3次/天 | 不限 | 不限 |
| 合盘 | 不限 | 不限 | 不限 |
| 云历史 | — | ✓ | ✓ |
| 付费前端 | ❌ | ❌ 无购买页面 | ❌ 无购买页面 |

- **支付 API 已实现** (`POST /api/payment`) 但仅有模拟支付
- **会员限制仅作用于 AI 解盘**，其他功能无限制
- **未登录用户无任何限流**

### API 路由

| 路由 | 方法 | 认证 | 说明 |
|------|------|------|------|
| `/api/generate` | POST | 否 | 排盘计算 |
| `/api/interpret` | POST | 可选 | AI 解盘 SSE（登录后限流） |
| `/api/heming` | POST | 否 | AI 合盘 |
| `/api/auth/register` | POST | 否 | 注册（手机号） |
| `/api/auth/login` | POST | 否 | 登录 |
| `/api/auth/me` | GET | 是 | 获取用户信息 |
| `/api/user/history` | GET/POST/DELETE | 是 | 排盘历史 CRUD |
| `/api/payment` | POST/GET | 是 | 支付/查询会员 |
